1

지구의 탄생과 지구시스템의 진화

성운 → 태양계 → 지구 → 4권역. "왜 그렇게 되는가"의 흐름으로.

1.1 태양계 형성 — 성운설

사슬: 초신성 충격파 → 수축·회전 → 납작한 원반 → 미행성체 → 원시행성 → 행성

응축(기체→고체 알갱이) · 집적(모여 쌓임) · 분화(내부에서 밀도순 층 분리). 수축의 최초 계기 = 초신성 충격파(중력 아님 — 중력은 그 후 계속 수축시키는 힘).

1.2 지구형 vs 목성형 최빈출

⚠️ 함정: 안쪽/바깥쪽 물질이 갈린 원인은 온도입니다. 밀도 차이는 원인이 아니라 결과!
구분크기밀도이유
지구형(수금지화)작음높음뜨거워 암석·금속만 응축
목성형(목토)큼낮음재료 많음 + 큰 중력으로 기체 포획
천왕성형(천해)중간—물·메테인·암모니아 얼음
개별 행성: 수성 = 3:2 공명(공전2=자전3) · 금성 = 자전(243일) > 공전(225일), 뜨거운 이유는 오직 CO₂ 온실효과 · 천왕성 = 자전축 98° 누움

1.3 태양계 외곽 구조

영역거리(AU)특징
카이퍼 벨트30~50도넛 원반, 단주기 혜성
산란 분포대50~100+해왕성에 차여 찌그러진 원반(에리스)
태양권계면~120태양풍 vs 성간풍 균형점
오르트 구름2천~10만구각, 장주기 혜성

성간 매질 = 수소·헬륨 99%. 논리: 멀다 → (케플러 3법칙) → 주기 길다 → 장주기 혜성.

1.4 운석 — "녹았는가(분화)"가 분류의 기준

운석 ├ 미분화(안 녹음, 콘드룰 있음) = 콘드라이트 │ ├ 오디너리(80%+, 철로 H/L/LL) │ ├ 엔스타타이트(지구와 가장 닮음, 동위원소 일치) │ └ 탄소질(차가운 외곽 → 물·유기물 → 생명 기원) └ 분화(녹음, 콘드룰 사라짐) ├ 에이콘드라이트(지각·맨틀) ├ 석철질(핵-맨틀 경계, 최희귀 1%) └ 철질(핵, 철-니켈)
핵심: 분화 = 모체가 녹음(용융) → 무거운 철은 핵, 가벼운 규산염은 맨틀. 크기는 "녹을 조건"일 뿐.
냉각 속도 대조: 콘드룰 = 빠르게 식음(작은 구슬) ↔ 비드만슈테텐 구조(철질) = 수백만~천만 년 느리게 식음 → "큰 소행성 핵"의 증거. 3·4단원 화산암(빠름)/심성암(느림)으로 또 나옴.

1.5.1 원시 지구 형성 순서 최빈출

⚠️ 순서 함정: 핵·맨틀 분리는 마그마 바다 다음! (녹기 전엔 분리 불가)
미행성체 충돌 → 원시 대기 → 마그마 바다 → 핵·맨틀 분리 → 원시 지각 → 원시 바다 → 생명체(40억 년) 암기: "충돌 → 대기 → 녹음 → 분리 → 굳음 → 바다"

마그마 바다 4대 에너지: 충돌열·중력수축열(외부) + 방사성붕괴열·분화열(내부).

1.5.2 기권의 진화 최빈출

대기 주성분: 수소·헬륨(날아감) → 수증기·CO₂(탈가스) → 질소(현재). 수증기는 바다로, CO₂는 석회암(지권)으로 빠지고 안정한 질소가 남음.

산소 등장 ★함정★ 남세균(35억 년) 광합성 → 산소가 먼저 바닷속 철과 반응 → 산화철 쌓여 호상철광층(BIF) ← 산소 혁명 "이전"! → 바다 철 다 소모 후 → 대기에 산소 축적(산소 혁명) → 오존층(6억 년) → 자외선 차단 → 육상 진출

1.5.3 태양 활동 플라즈마가 열쇠

플라즈마 = 이온화 기체 + 전기 전도성 + 자기장에 반응(← 모든 태양 현상의 원인). 자기장은 플라즈마 대류=전류 → (앙페르) → 자기장.

흑점 ★함정★: 어두운 이유 = 온도 낮음. 왜 낮은가 = 강한 자기장이 플라즈마 대류를 억제 → 열 공급 차단. (열을 직접 막는 게 아니라 대류를 막음)
태양풍 ★함정★: 전자 + 양성자 (중성미자 아님! 빛도 아님 — 질량 가진 하전 입자 흐름)

플레어 = 자기장 폭발(델린저 현상) · 홍염 = 자기장 고리 따라 아치 · 활동주기 11년(흑점 수 + 자기장 극성 반전).

1.5.4 지구 자기장·밴앨런대·오로라

다이너모 이론: 액체 외핵 대류 → 전류 → 자기장. 태양과 원리 같음, 매질만 다름(플라즈마 vs 액체 금속).

2단 방어 ★함정★: 자기장이 오존층을 지키고, 오존층이 자외선을 막는다. 자기장이 자외선을 직접 막는 게 아님. (사례: 화성)

내대1천~1.2만 km양성자
외대1.3만~6만 km전자

💚 초록산소100~200km(최빈)

지자기극(이론) ≠ 자극(실제 측정, 복각 90°). 원인 = 비쌍극자 자장. 쌍극자 축은 자전축과 11° 기울어짐. 편각=좌우(진북vs자북), 복각=위아래(위도 0°→90°).

1.6 지구시스템 4권역 + 외권

상호작용 함정: 두 권역으로 단정 금지. 예) "바다에 녹은 CO₂ → 조개 껍데기 → 석회암" = 수권↔생물권↔지권(3개!)

2

판구조론과 플룸구조론

과학사 + 메커니즘. "각 이론이 앞 이론의 무엇을 해결했나"로.

2.1 이론 진화 사슬 뼈대

대륙이동설(베게너,1915) "증거는 있는데 원동력이 없다" → 맨틀대류설(홈스,1928) "원동력은 맨틀 대류다" → 해저확장설(헤스,1961) "그 증거를 찾았다" → 판구조론(모건,1967) "전부 종합" → 플룸구조론(1990s) "열점·맨틀대류까지 설명"
⚠️ 순서 함정: 맨틀대류설(1928)이 해저확장설(1961)보다 먼저. 원인을 먼저 제안, 증거는 나중.

2.2 대륙이동설 (베게너)

외면당한 이유 ★핵심★: 원동력(대륙 움직인 힘)을 설명 못 함. 증거는 많았지만 "무슨 힘?"에 답 못 해서.

증거 4갈래(공통 논리: 지금 떨어진 게 과거엔 연속): ① 해안선(수심 200m) ② 화석(메소사우루스·글로소프테리스·리스트로사우루스) ③ 빙하 흔적(적도에 빙하 자국) ④ 지질구조(애팔래치아↔칼레도니아).

2.3 고지자기 — 부활 계기 어려움

암석이 굳을 때 자기장 방향이 화석화된 나침반처럼 기록 → 유럽·북미 자극 경로가 둘로 갈라짐 → 자극은 하나뿐이니 움직인 건 대륙 → 대륙 되돌리면 경로가 겹침 = 과거 한 덩어리 증거(겉보기 자극 이동 경로).

2.4 해저확장설

"해령에서 멀어질수록": 나이↑(최대 2억 년 ≪ 대륙 39억 년) · 퇴적물 두께↑ · 열류량↓ · 자기 줄무늬 = 해령 축 좌우 대칭.

자기 줄무늬 대칭: 지자기 역전(1단원!)이 반복 기록 + 양쪽 동시·같은 속도 확장 → 거울상 대칭. 해저 확장의 결정적 증거.

2.5 판구조론 — 성립 가정

① 판 아래 무른 연약권 존재 ② 암석권은 단단한 강체 판. 암석권(판) = 지각 + 맨틀 최상부(0~100km) ≠ 지각만! 대륙판(가벼움) vs 해양판(무거움) → 밀도 차가 섭입 방향 결정.

2.6 판의 경계 ★★★시험 핵심★★★

경계힘지각지형단층지진마그마
발산장력생성해령·열곡대정단층천발만현무암질
섭입횡압력소멸해구역단층천발~심발안산암질
충돌횡압력—습곡산맥역단층천발~중발거의 없음
보존어긋남보존변환단층주향이동천발만없음
발산 천발만 = 섭입(가라앉는 판) 없어 지진원 얕음. 마그마=현무암질(맨틀 직접), 해저선 베개용암. 사례 동아프리카 열곡대 → 미래 홍해처럼 바다.

섭입 = 베니오프대(해구→대륙쪽 지진 깊어짐). 물 → 녹는점 하강 → 부분 용융 → 안산암질(해양 현무암질+대륙 화강암질 혼합).

충돌(히말라야) = 둘 다 가벼워 못 가라앉고 솟음, 화산 거의 없음.

보존(산안드레아스) = 어긋나 스침, 화산X·천발만.

2.7 판 이동 원동력 (3가설)

① 해령에서 미는 힘 ② 섭입판이 당기는 힘 ③ 경사진 연약권 위를 미끄러지는 힘.

2.8 플룸구조론 (1990s)

지진파 토모그래피로 등장. 판구조론이 못 푼 맨틀 대류·열점 설명. 차가운 플룸(슬래브 낙하) ↔ 뜨거운 플룸(반작용 상승, 초대륙 분열). 열점=플룸이 만든 화산(하와이), 판 움직여도 고정.

마그마와 화성암

2단원 판 경계 + 1단원 분화가 여기서 합쳐집니다.

3.1 마그마 종류 (생성 위치로 결정)

3.2 마그마 생성 3방식 중요

온도 상승(방사성 붕괴열·열점) · 압력 하강(중앙해령 감압 용융) · 물 공급(함수광물이 물 방출 → 녹는점 하강 → 섭입대).

4

화성암의 조직과 편광현미경

광물의 광학적 성질 — 수행평가·실험 단골.

4.1 지각 8대 원소 암기

O > Si > Al > Fe > Ca > Na > K > Mg "오 시 알 페 카 나 크 마" (O 46%, Si 28%)
⚠️ 주의: 지각은 O 1위지만, 지구 전체는 철(Fe) 1위, 산소 2위.

4.2 광물의 정의

자연산 + 무기물 + 일정한 화학성분 + 규칙적 결정구조 + 고체. 광물 아님: 인공 다이아몬드, 물, 액체 수은, 석유. 무색광물(Fe·Mg 없음: 석영·장석) vs 유색광물(Fe·Mg 있음: 감람석·휘석·각섬석·흑운모).

4.3 편광현미경 핵심

박편 두께 = 0.03mm(고정 — 두께 따라 간섭색 변하므로).

모드관찰 가능
개방니콜고유색, 다색성, 투명성, 입자크기, 벽개
직교니콜간섭색, 완전소광, 회전소광, 쌍정
광축굴절직교니콜 회전 시
등방체없음단굴절완전소광(계속 어두움)
이방체있음복굴절간섭색 변화, 360°에 4회 소광

상광선(광축 수직, 스넬 따름, 회전해도 안 변함) vs 이상광선(광축 평행, 스넬 안 따름, 회전 시 변함). 간섭색=상·이상광선 속도차→위상차→간섭.

4.4 조암광물 광학 성질 암기표

광물색다색성쪼개짐쌍정

파동소광(석영)=변성 증거 · 누대구조(사장석)=연속반응계열.

5

지구 내부 구조

지진파로 들여다본 지구 — 1단원 분화의 최종 결과물.

5.1 지진파 핵심

종류통과 매질속도피해
P파종파고체·액체·기체 모두빠름작음
S파횡파고체만중간큼
표면파—지표면가장 느림가장 큼

S파가 고체만 = 액체는 전단계수 μ=0 → Vs=0. P파는 액체서 느려지지만 0 아님(K≠0). 표면파: 러브파(수평)·레일리파(수직 타원).

5.2 지진 깊이 분류 (2단원 직결)

깊이어디서
천발~70km모든 판 경계, 최빈·피해 큼
중발70~300km충돌형·섭입형
심발300~700km섭입형 베니오프대만

5.3 지진 세기

규모진도
의미진원 에너지받는 영향
위치어디서나 동일위치마다 다름

핵실험/발파 = P파 강·S파 약(P/S 비로 판별). 유발지진(2017 포항) = 단층운동이라 S파 뚜렷.

5.4 불연속면 최빈출

불연속면깊이경계근거
모호면~35km지각/맨틀주시곡선 기울기 급변
구텐베르크면2,900km맨틀/외핵P파 암영대(103~142°), S파 도달X → 외핵 액체
레만면5,100km외핵/내핵암영대에 약한 P파 → 내핵
⚠️ 외핵 액체 증거 = S파 암영대(S파가 외핵 통과 못 함). 1단원 다이너모(액체 외핵)와 직결!

5.5 지구 내부 구조 — 분화의 최종 결과물

지각 (규산염, 가벼움) — 대륙(화강암질,밀도2.7) / 해양(현무암질,밀도3.0) 맨틀 (감람암질, 부피 82%) — 저속도층·전이대·하부맨틀 외핵 (철·니켈·황, 2900~5100km, 액체, S파 X) 내핵 (철·니켈, 5100~6400km, 고체, 압력으로 안 녹음)
외핵 액체 / 내핵 고체 — 같은 철·니켈인데 깊을수록 압력이 녹는점을 올려 내핵은 못 녹음. 위로 갈수록 가볍고 아래로 무겁다 = 1단원 마그마 바다 분화의 결과.

5.6 토모그래피 (2단원 플룸과 연결)

지진파 속도로 내부 온도 영상화. 느린 곳=뜨거움(뜨거운 플룸), 빠른 곳=차가움(차가운 플룸). 플룸 존재를 이걸로 확인.

★단원을 가로지르는 핵심 연결 (이것만은!)

  1. 분화 = 녹아서 밀도순 분리 → 1단원(핵·맨틀) = 3단원(마그마 분화) = 5단원(층 구조). 세 번 반복.
  2. 액체 외핵 → 1단원(다이너모로 자기장) + 5단원(S파 암영대로 증명). 양방향.
  3. 물이 녹는점을 낮춤 → 2단원(섭입대) = 3단원(함수광물). 똑같은 메커니즘.
  4. 냉각 속도가 조직 결정 → 1단원(콘드룰/비드만슈테텐) = 3·4단원(화산암/심성암).
  5. 지자기 역전 → 1단원(자기장 반전) → 2단원(해저 자기 줄무늬 대칭).
  6. 현무암질=발산, 안산암질=섭입 → 2단원 판 경계 = 3단원 마그마.
  7. 베니오프대·천발~심발 → 2단원(섭입경계) = 5단원(지진 깊이).
  8. 밀도 → 행성(1)·섭입(2)·결정 침강(3)·지구 층(5) 전부 결정.

Q퀴즈 13문항 — 먼저 답하고 펼쳐서 확인

Q1지구형·목성형 행성의 크기·밀도 차이를 만든 근본 원인은? "밀도 차이"는 원인인가 결과인가?
온도. 성운이 바깥으로 갈수록 차가워, 안쪽은 암석·금속만 응축(작고 무거움), 바깥쪽은 얼음·기체까지 응축(크고 가벼움). 밀도 차이는 원인이 아니라 결과.
Q2원시 지구 형성 순서를 쓰세요.
미행성체 충돌 → 원시 대기 → 마그마 바다 → 핵·맨틀 분리 → 원시 지각 → 원시 바다. 핵·맨틀 분리는 마그마 바다 다음(녹어야 분리).
Q3호상철광층(BIF)은 산소 혁명 이전인가 이후인가? 이유는?
이전. 남세균이 만든 산소가 먼저 바닷속 철을 산화시켜 BIF를 쌓고, 바다 철이 다 소모된 뒤에야 대기에 산소가 축적(산소 혁명)되기 때문.
Q4흑점이 어두운(온도 낮은) 진짜 이유를 메커니즘으로.
흑점은 자기장이 비정상적으로 강한 곳 → 강한 자기장이 플라즈마의 대류를 억제 → 아래에서 올라오는 열 공급이 차단 → 온도 하강 → 어둡게 보임. (열을 직접 막는 게 아니라 대류를 막음)
Q5베게너의 대륙이동설이 외면당한 결정적 이유는?
대륙을 움직인 원동력(힘)을 설명하지 못해서. 증거는 많았지만 메커니즘이 없어 받아들여지지 않음. → 홈스의 맨틀대류설이 채움.
Q6시간 순서로: 해저확장설 / 판구조론 / 맨틀대류설 / 대륙이동설
대륙이동설(1915) → 맨틀대류설(1928) → 해저확장설(1961) → 판구조론(1967). 맨틀대류설이 해저확장설보다 먼저!
Q7판 경계별 단층/지진 깊이/화산을 쓰세요. (발산/섭입/충돌/보존)
발산: 정단층·천발만·현무암질. 섭입: 역단층·천발~심발·안산암질. 충돌: 역단층·천발~중발·화산 거의 없음. 보존: 주향이동·천발만·화산 없음.
Q8섭입대 마그마 생성 메커니즘은? 3단원의 무엇과 같은가?
섭입 해양판의 물(함수광물)이 맨틀의 녹는점을 낮춤 → 부분 용융 → 마그마. 3단원 "물 공급에 의한 마그마 생성"과 같은 메커니즘.
Q9현무암질·안산암질·유문암질 중 (a)SiO₂ 최다 (b)점성 최대 (c)가장 폭발적인 것? (셋 다 같은 답)
유문암질. SiO₂ 많을수록 점성 크고 폭발적이며 온도는 낮음.
Q10직교니콜에서 회전 시, 광학적 등방체와 이방체는 각각 어떻게 보이나?
등방체: 계속 완전소광(어두움). 이방체: 간섭색이 변하고 360° 회전에 4회 소광.
Q11휘석과 각섬석의 쪼개짐 각도는?
휘석 약 90°, 각섬석 약 120°. (다색성은 각섬석이 더 뚜렷)
Q12외핵이 액체라는 증거는? 1단원의 무엇과 연결되나?
S파 암영대 — S파(고체만 통과)가 구텐베르크면 아래로 못 가서 도달하지 않음 → 외핵 액체. 1단원 다이너모 이론(액체 외핵 대류로 자기장 생성)과 연결.
Q13P파와 S파가 각각 통과하는 물질 상태는?
P파: 고체·액체·기체 모두. S파: 고체만.
