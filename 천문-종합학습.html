06

별의 물리량

별빛에서 온도·광도·크기를 읽어내는 법.

1. 별의 색과 표면 온도 — 흑체 복사

흑체: 입사 에너지를 모두 흡수·방출하는 이상 물체. 별은 거의 흑체라서, 흑체로 가정해 온도·광도를 추정합니다.

슈테판·볼츠만 법칙
E = σT⁴ — 단위 넓이가 단위 시간에 방출하는 에너지는 표면 온도의 4제곱에 비례. (온도 2배 → 에너지 16배)
빈의 변위 법칙
λ_max = a / T — 온도가 높을수록 최대 에너지 파장이 짧아짐(파랗게). 온도와 반비례.
플랑크 곡선: 온도 높을수록 모든 파장에서 더 많은 에너지 방출(곡선이 통째로 위로) + 최대 파장은 왼쪽(짧은 쪽)으로. 곡선 아래 넓이 = 총 방출 에너지(=σT⁴).

2. 분광형 — 흡수선으로 온도 분류

흡수 스펙트럼: 별빛이 별의 대기를 지나며 특정 파장이 흡수돼 검은 선(흡수선) 발생. 별의 구성 성분은 비슷한데 스펙트럼이 다른 이유 = 표면 온도가 달라 원소의 이온화 정도가 다르기 때문.

3. 광도와 밝기

광도(L) = 별이 단위 시간에 표면에서 방출하는 총 에너지. 표면적 × σT⁴.

광도 공식
L = 4πR² · σT⁴ — 표면적(4πR²) × 단위넓이 방출량(σT⁴). → 광도·온도를 알면 반지름 R을 역산.

07

H-R도와 별의 진화

온도·광도를 한 평면에 찍으면 별의 일생이 보입니다.

1. H-R도 — 가로 온도, 세로 광도

헤르츠스프룽 + 러셀. 가로축 = 분광형/표면 온도(왼쪽이 고온), 세로축 = 절대 등급/광도(위가 밝음).

종류위치특징
주계열성좌상→우하 대각선별의 80%+. 온도↑일수록 광도·반지름·질량 큼
거성주계열 오른쪽 위온도 낮은데 광도 큼 → 반지름 큼
초거성거성보다 위반지름 매우 큼 → 밀도 작음
백색왜성왼쪽 아래온도 높은데 광도 작음 → 반지름 작고 밀도 큼
평균 밀도: 백색왜성 > 주계열성 > 거성 > 초거성. 광도 계급(MK 분류): 태양 = G2V(G2 + 주계열 V).

2. 별의 에너지원

중력 수축 에너지: 수축하며 위치E → 운동E(내부 온도↑) + 복사E(빛). 핵융합 에너지: 질량 결손이 에너지로(E=mc²).

핵융합 순서: 수소 → 헬륨 → 탄소 → 네온 → 산소 → 규소 → 철(마지막) 수소핵융합: 1000만K↑ / 헬륨핵융합(3α): 1억K↑

3. 별의 진화 — 질량이 운명을 결정 최빈출

원시별(전주계열, 핵융합 X) → 주계열(수소 핵융합 시작, 일생 80~90%, 정역학 평형) → 적색거성/초거성(중심 수소 소진 → 수축 → 외곽 팽창) → 행성상성운 + 백색왜성 (태양급) → 초신성 → 중성자별/블랙홀 (대질량)
⚠️ 주계열 기간 = 질량 클수록 짧다! (직관과 반대 — 연료 많아도 훨씬 빨리 태움) 태양급 = 약 100억 년. 질량 큰 별일수록 좌상단, 수명 짧음.

08

가장 가까운 별, 태양

1단원 흑점이 여기서 더 깊어집니다 — 내부·표면·대기·활동.

1. 태양이 중요한 이유 · 내부 구조

태양계 질량의 99.9%, 에너지 대부분의 근원, 가장 가까운 항성(다른 별 연구의 기초). 중심핵에서 수소 핵융합(주계열성).

내부(중심→밖): 핵(~0.3R) → 복사층(~0.7R) → 대류층 → [광구] 핵=에너지 생성 / 복사층=복사로 전달 / 대류층=대류로 전달

2. 태양의 표면(광구)

광구: 눈에 보이는 표면, 태양 복사 대부분이 여기서 방출. 쌀알무늬: 광구 아래 대류 때문(밝은 중심=상승, 어두운 가장자리=하강·냉각). 지름 ~1000km, 수명 5~수십 분.

흑점 ★핵심★: 강한 자기장이 대류에 의한 에너지 전달을 방해 → 주변보다 온도 낮아(~3,800K) 검게 보임. 크기 ~10,000km. 암부(중심·가장 어두움) + 반암부.
차등 자전 ★중요★: 흑점으로 측정한 자전 주기 = 적도 25일, 중위도 27일, 극 35일. 고위도로 갈수록 자전 주기 길다. 태양이 고체가 아니라서.
자기 활동 주기 = 22년 ★함정★: 흑점 수 극대·극소는 11년 주기지만, 새 주기 흑점은 이전과 극성이 반대 → 극성까지 제자리로 돌아오려면 22년이 진짜 활동 주기. 흑점은 중위도→저위도로 이동(나비도).

3. 태양의 대기 — 위로 갈수록 뜨거워짐

층특징
채층하부 대기(광구 위 ~1만km), 4500→6000K로 상승, 수소 알파선(656.3nm)으로 분홍색, 개기일식 때만
코로나상부 대기, 밀도 매우 작음, 약 100만K(매우 뜨거움) → 자외선·X선 방출, 개기일식 때 진줏빛

채층 현상: 스피큘(채층 위 솟는 가스, 수명 15분↓) · 홍염(채층→코로나 뻗는 가스, 원반에선 필라멘트로 보임).

코로나에서 나오는 것 — 태양풍(고온이라 입자가 중력 탈출, 양성자·전자, 지구 부근 ~400km/s) · 플레어(폭발, X선·자외선이 빛 속도로 와 이온층 교란 → 전파 통신 장애) · CME(코로나 물질 분출). 모두 흑점 극대기에 활발.
⚠️ 코로나 역설: 코로나는 광구(5800K)보다 훨씬 뜨거운데(100만K), 입자 개수 밀도는 매우 낮음. "뜨겁다 = 입자가 많다"가 아님.
09

외계 행성과 생명체 탐사

별빛의 미세한 변화로 보이지 않는 행성을 찾습니다.

1. 외계 행성 탐사 4가지 방법 최빈출

방법원리한계 / 발견 경향
시선 속도(도플러)별·행성이 공통 질량중심 공전 → 별 스펙트럼 적색/청색 이동 반복궤도면이 시선에 수직이면 불가. 질량 큰 행성 발견 쉬움
식 현상(통과)행성이 별 앞 통과 → 별 밝기 감소. 감소량=크기비, 주기=공전주기궤도면이 시선과 나란해야 함. 공전 반지름 작은 행성
미세 중력 렌즈앞 별의 중력으로 뒤 별빛 휨 + 행성 중력으로 밝기 변화 한 번 더궤도면 무관·발견 가능, 그러나 주기 관측 불가(1회성). 반지름 큰 행성
직접 촬영중심별 빛 가리고 행성 직접 촬영중심별과 멀고 밝은 행성만. 거리 먼 외계행성
시선 속도 핵심: 행성 질량 클수록 공통 질량중심이 행성 쪽으로 치우쳐 → 중심별 공전 속도↑ → 도플러 효과 큼 → 발견 쉬움. 시선 속도 최댓값 = v·cosθ(θ=궤도면과 시선의 각, 나란할수록 큼).

2. 생명 가능 지대 (HZ)

가장 중요한 요소 = 액체 상태의 물. (비열 커 항상성 유지 + 극성이라 좋은 용매). 생명 가능 지대 = 물이 액체로 존재하는 영역.

HZ 위치·폭
전달 에너지 ∝ 광도, ∝ 1/거리². → 광도 클수록 HZ가 멀어지고 넓어짐.
중심별생명 가능 지대 특징
질량 큰 별수명 짧음 → 생명 진화 시간 부족, 광도 변화 커 HZ 변동 큼
질량 작은 별HZ가 가깝고 좁음 → 행성이 들 확률 낮음, 너무 가까우면 동주기 자전(낮밤 없음)
⚠️ HZ에 있어도 생명체 보장 아님: 추가로 적당한 두께의 대기 + 고에너지 입자를 막을 자기장이 필요. (1단원 "자기장이 생명 보호"와 연결!)

★단원·과목을 가로지르는 연결

  1. 온도 → 색 → 분광형 → 06(빈 법칙·분광형) = 07(H-R도 가로축) = 08(태양 G2).
  2. 질량이 운명 결정 → 07(주계열 기간·최종 단계) = 09(생명 가능 지대 폭·수명).
  3. 흑점 = 자기장이 대류를 막아 온도↓ → 고체지구 1.5.3 = 천문 08. 똑같은 메커니즘, 08이 더 깊음(차등 자전·22년).
  4. 태양풍·플레어·CME → 고체지구 1.5.3(태양 활동) = 천문 08(코로나). 전자·양성자 흐름.
  5. 초신성이 무거운 원소를 흩뿌림 → 07(대질량 별 최후) = 고체지구 1.1(성운 수축 계기).
  6. 자기장이 생명 보호 → 09(HZ 추가 조건) = 고체지구 1.5.4(2단 방어).
  7. 플라즈마 → 08(광구·태양풍) = 고체지구 1.5.3. 이온화 기체, 자기장 반응.
  8. 도플러 효과 → 09(시선 속도법). 적색이동=멀어짐, 청색이동=가까워짐.

Q퀴즈 12문항 — 먼저 답하고 펼쳐서 확인

Q1표면 온도가 2배인 흑체는 단위 넓이당 방출 에너지가 몇 배? 최대 복사 파장은?
에너지 16배(σT⁴, 2⁴). 최대 파장은 ½배(빈 법칙, T에 반비례).
Q2별마다 구성 성분은 비슷한데 흡수 스펙트럼이 다른 이유는?
표면 온도가 달라서. 온도에 따라 원소의 이온화 정도가 다르고, 이온화 단계마다 특정 흡수선을 만들어 흡수선의 종류·세기가 달라짐.
Q3분광형 순서와, 가장 뜨거운 형·태양의 분광형은?
O B A F G K M(고온→저온, 파랑→빨강). 가장 뜨거운 건 O형, 태양은 G2(약 5800K). A형은 수소 흡수선 최강.
Q4광도가 태양의 16배, 표면 온도가 태양의 0.5배인 별의 반지름은 태양의 몇 배?
L=4πR²σT⁴ → R ∝ √L / T². R = √16 / 0.5² = 4 / 0.25 = 16배.
Q5절대 등급과 겉보기 등급은 각각 무엇에 영향받나?
절대등급: 10pc 거리 가정한 실제 밝기 → 광도만. 겉보기등급: 거리 + 광도. 둘 다 작을수록 밝음.
Q6H-R도에서 거성과 백색왜성의 밀도를 비교하면? 주계열성은 어디에 분포?
밀도 백색왜성 ≫ 거성(전체 순: 백색왜성>주계열>거성>초거성). 주계열성은 좌상→우하 대각선, 별의 80%+.
Q7주계열 단계에 머무는 기간은 질량이 클수록 길까 짧을까? 이유는?
짧다. 질량 크면 연료는 많지만 중심 온도가 높아 핵융합이 훨씬 격렬해 연료를 빠르게 소비. (태양급 ≈ 100억 년)
Q8초기 질량에 따른 별의 최종 단계 3가지는?
~10 태양질량 = 백색왜성, 10~40 = 중성자별, 40↑ = 블랙홀. 철보다 무거운 원소는 초신성 폭발에서 생성.
Q9흑점이 주변보다 어두운(온도 낮은) 이유는?
강한 자기장이 대류에 의한 에너지 전달을 방해 → 열 공급 차단 → 온도 낮아짐(~3,800K) → 어둡게 보임.
Q10태양의 자기 활동 주기가 11년이 아니라 22년인 이유는?
흑점 수 극대·극소는 11년 주기지만, 새 주기 흑점은 이전 주기와 극성이 반대. 극성까지 원래대로 돌아오려면 11×2 = 22년이 진짜 주기.
Q11식 현상(통과)법과 시선 속도(도플러)법은 각각 어떤 행성을 발견하기 쉬운가? 공통 한계는?
식 현상: 공전 반지름 작은(가까운) 행성. 시선 속도: 질량 큰 행성. 공통 한계 — 둘 다 궤도면이 시선과 나란해야(수직이면 불가). 미세 중력 렌즈는 궤도면 무관.
Q12행성이 생명 가능 지대에 있으면 반드시 생명체가 존재하는가?
아니다. 물이 액체일 조건은 충족하지만, 추가로 적당한 두께의 대기와 고에너지 입자를 막을 자기장이 필요. (고체지구 자기장 단원과 연결)
